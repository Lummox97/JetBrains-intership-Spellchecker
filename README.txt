Спеллчекер строится на допущении, что человек может допустить всего четыре типа ошибок:
1) Пропущена буква
2) Соседние буквы перепутаны местами
3) Вставлена лишняя буква
4) Перепутана одна буква

Функция edits генерирует все возможные слова с этими ошибками из слова введённого, пользователем, а затем они сравниваются со словами в словаре.

Поле depth отвечает за количество ошибок, допущенных во введённом слове. Стандартно оно настроено на два, три - медленно на длинных словах, четыре - на всех.

Словарь можно подключить любой: файл - Dict.txt, слова разделены пробелами, исправление опечаток и пропуска букв работает только на латинском алфавите.
